\doxysubsection{Detailed Description}
Represents a processing unit in the Milli\+Suono graph. 

A \doxylink{classms_1_1_node}{Node} defines a functional unit (e.\+g. an oscillator, filter, or mixer) with a unique identifier and a set of configurable parameters. Nodes can be connected via Ports to form complex audio processing graphs.
